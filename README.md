# homebrew-bopca

Homebrew tap for [bopca](https://github.com/cboone/bopca). Run LLM agents in lightweight macOS VMs.

## Installation

```bash
brew install cboone/bopca/bopca
```

Or tap first, then install:

```bash
brew tap cboone/bopca
brew install bopca
```
